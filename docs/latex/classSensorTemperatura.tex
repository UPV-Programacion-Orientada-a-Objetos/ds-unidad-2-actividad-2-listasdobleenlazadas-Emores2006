\doxysubsection{Descripción detallada}
Clase derivada que maneja lecturas de temperatura en punto flotante. 

Implementa la funcionalidad específica para sensores de temperatura, procesando lecturas de tipo float y calculando promedios tras eliminar el valor más bajo.
